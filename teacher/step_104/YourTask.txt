Клиент нанял команду тестировщиков для тестирования нашего
приложения. Клиент хочет, чтобы каждый тестировщик запускал
и тестировал приложение на своём локальном компьютере.
У тестировщиков возникли проблемы с установкой MySQL базы
данных на локальные компы. Клиент обратился к нам с просьбой
о том, чтобы мы собрали версию приложения с in memory базой
данных, чтобы тестировщикам не нужно было устанавливать MySQL
сервер и MySQL Workbench на локальные компьютеры. Клиент слышал,
что подход с in memory базой данных мы используем в тестах.

Ментор просит вас изучить этот вопрос и перевести приложение
на использование in memory базы данных и собрать новую версию
приложения для команды тестировщиков.


Ваша задача:

С помощью интернета разобраться как перевести приложение
на H2 in memory базу данных, и найти способ предоставить к ней доступ
для написания SQL запросов.

PS: https://www.baeldung.com/spring-boot-h2-database
