В этом задании вы научитесь собирать (билдить)
проект с помощью команд из закладки Gradle.

Для начала давайте проверим настройки проекта.
Откройте меню: File -> Project Structure
и убедитесь, что у вас выбрана
SDK 18 (например Amazon Corretto version 18.0.2)
и установлен Language Level: 18 - JavaDoc snippets.

Откройте меню: File -> Settings и в верхнем левом углу
в строке поиска наберите Gradle. Убедитесь, что
Gradle JVM: установлена в Project SDK corretto 18.
В этом же окне установите:
Build and run using: IntelliJ IDEA
Run tests using: IntelliJ IDEA

В данном проекте для сборки проекта используется Build Tool Gradle.
Его закладку вы найдёте справа в IntelliJ IDEA. Откройте ее.

Важная функциональность в Gradle закладке это символ с двумя стрелочками
Reload all Gradle projects - обязательно используйте его после добавления
в проект новых зависимостей.

Запустить полную сборку проекта можно выполнив команду
Tasks -> build -> build в окне Gradle. Эта команда запустит
сборку всех модулей в проекте. Выполнение этой команды может
занять определённое время, так как модулей в проекте достаточно много.

Запустите полную сборку проекта!

Для того чтобы выполнить сборку только вашего модуля в окне Gradle
выберите название вашего модуля и запустите команду:
student_lastname_firstname -> Tasks -> build -> build.
(используйте вместо student_lastname_firstname название вашего модуля).

Запустите сборку вашего модуля!
