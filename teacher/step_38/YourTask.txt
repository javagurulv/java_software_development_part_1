Ваша задача: запустить приложение и послать к нему HTTP запрос
для расчёта стоимости страхового полиса. Это продемонстрирует
вам то, как будут пользоваться разрабатываемым вами сервисом.

Для создания и посылки HTTP запросов воспользуйтесь программой
Postman (https://www.postman.com/downloads/).

После установки Postman создайте HTTP POST запрос и заполните
секции Headers и Body по примеру приведённому в файлах:
Postman_request_headers.png
Postman_request_body.png

Body JSON example:
{
"personFirstName" : "Vasja",
"personLastName" : "Pupkin",
"agreementDateFrom" : "2021-05-25",
"agreementDateTo" : "2021-05-29"
}

У вас будет много вопросов!
Воспользуйтесь помощью ментора и силой интернета!

PS: создайте как валидные так и не валидные запросы и убедитесь
в том, что ваш сервис выдаёт ошибки при получении не валидного запроса.

PS: в рамках этого задания познакомьтесь с протоколом HTTP.
Это один из главных протоколов уровня приложения (application level)
используемых в интернете. При изучении информации про HTTP особенно
обратите внимание на разные типы запросов (GET/POST/etc.), которые
можно посылать при помощи этого протокола.

PS: не забудьте по окончанию выполнения шага создать и заполнить
файл /documentation/Step_X_description.txt. Формат заполнения этого
файла описан в step_1 в файле Learning_reflection.txt.
