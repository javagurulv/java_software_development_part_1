Ваша задача:

1. Создайте параметр в application.properties
указывающий путь выгрузки файлов.

2. Реализуйте выгрузку агриментов в xml файлы по схеме:
- загружаем из базы uuids всех агриментов
  используя для этого команду из предыдущего шага;
- по одному обрабатываем каждый uuid агримента;
- достаем из базы данные по uuid агрименту используя
  для этого уже существующую команду;
- создаем xml;
- сохраняем в файл.
