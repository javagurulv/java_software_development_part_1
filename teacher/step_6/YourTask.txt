В этом задании вы научитесь удалять артефакты (файлы)
созданные Gradle в процессе сборки приложения.

В папке build расположены артефакты созданные Gradle в процессе сборки приложения.

У Gradle есть полезная команда, которая находится рядом
с командой build. Это команда clean. Эта команда удаляет все
артефакты созданные Gradle во время сборки. Эти артефакты можно
увидеть в папке build (обычно подсвечивается красным в IDEA).

По аналогии с командой build, запустите команду clean из закладки Gradle
для всего проекта и только для вашего модуля.

По аналогии с командой build, запустите команду clean из терминала
для всего проекта и только для вашего модуля.

./gradlew clean
./gradlew :student_lastname_firstname:clean

Команда Gradle clean удаляет папку build, то есть удаляет все артефакты
созданные в процессе сборки приложения.
