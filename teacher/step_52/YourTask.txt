Одним из важных параметров обработки запросов является время.
Сколько времени приложение тратит на обработку каждого запроса?

Ментор предлагает добавить в проект функционал замера
времени выполнения запроса. Для этого предлагается
воспользоваться классом Stopwatch из библиотеки Google Guava.

Ваша задача: замерить и залогировать время выполнения запроса.

Примерный ожидаемый результат на консоли:
2023-03-13 08:15:44,682 [http-nio-8080-exec-1] INFO  l.j.j.i.r.TravelCalculatePremiumRequestExecutionTimeLogger - Request processing time (ms): 70
