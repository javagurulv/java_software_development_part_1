Архитектура приложения это важная часть разработки программного продукта.

Для ее визуализации и документации используется нотация C4.

Ваша задача: изучить материалы про нотацию C4:
https://c4model.com/
https://habr.com/ru/articles/778726/
https://habr.com/ru/companies/nspk/articles/679426/

Попробуйте изучить и воспользоваться
инструментом для создания С4 диаграмм:
https://www.structurizr.com/dsl.

Попробуйте нарисовать для нашего приложения:
- C4 Context diagram
- C4 Container diagram
- C4 Component diagram

Созданные вами диаграммы поместите в папку /documentation/c4_diagrams.

Цель этого задания познакомить вас с нотацией C4 используемой
в описании и визуализации архитектур программных продуктов.
