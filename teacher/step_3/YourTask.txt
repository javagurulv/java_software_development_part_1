В этом задании вы научитесь собирать (билдить) ваш проект
с помощью команд Gradle из терминала.

Gradle команды можно запускать так же из консоли (терминала).
Для этого откройте закладку Terminal (обычно находится
в нижней панели инструментов в среде разработки).
В открывшемся окне напишите команду:

./gradlew build

и нажмите Enter. Это запустит полную сборку проекта.

Если проект большой и полная сборка проекта выполняется очень долго,
то удобнее запускать только сборку своего модуля. Для этого воспользуйтесь
следующей командой:

./gradlew :student_lastname_firstname:build

Эта команда запустит сборку только для модуля student_lastname_firstname.
Попробуйте подставить вместо student_lastname_firstname название
своего модуля.

PS: не забудьте по окончанию выполнения шага создать и заполнить
файл /documentation/Step_X_description.txt. Формат заполнения этого
файла описан в step_1 в файле Learning_reflection.txt.
