Ментор предлагает следующий план реализации AgeCoefficient для
медицинского риска:

1. Добавить в запрос дату рождения человека по которой и будет
определяться его возраст.
2. Добавить в ответ дату рождения человека.
3. Создать в базе данных таблицу AGE_COEFFICIENT с полями:
age_from, age_to, coefficient.
4. Заполнить таблицу AGE_COEFFICIENT данными.
5. Создать в Java доменный объект + репозиторий для написания
запросов к таблице AGE_COEFFICIENT.
6. Поменять формулу расчёта медицинского риска на:
premium = CountryDefaultDayRate * DayCount * AgeCoefficient.

Понятен ли вам план предлагаемый ментором, обсудите план с ментором,
если есть вопросы - проясните их.
